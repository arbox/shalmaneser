compute context features: for each context in the given list of context sizes, compute a context with feature_type “CXNN” (where NN is the size of the context) and with features wordlemma#pos#ne

enter each context into the feature hash 

     # File FredFeaturize.rb, line 506
506:   def compute_context_features(context,  # array: word*lemma*pos*ne
507:                                center_pos, # int: context is 2*cx_size_onesided + 1 long
508:                                context_sizes,    # array:int, produce a context of each of these sizes
509:                                feature_hash)     # hash: feature_type -> array:feature, enter features here
510:     
511:     
512:     context_sizes.each { |context_size|
513:       # feature type: CXNN, where NN is the size of the context
514:       feature_type = "CX" + context_size.to_s()
515:       
516:       # features: an array of strings
517:       feature_hash[feature_type]  = Array.new()
518:       
519:       # pre-context
520:       (center_pos - context_size).upto(center_pos - 1) { |ix|
521:         if context[ix]
522:           # context entries may be nil at the beginning and end of the text
523:           feature_hash[feature_type] << context[ix].map { |e| e.to_s() }.join("#").gsub(/\s/, "_")
524:         end
525:       }
526:       # post-context
527:       (center_pos + 1).upto(center_pos + context_size) { |ix|
528:         if context[ix]
529:           # context entries may be nil at the beginning and end of the text
530:           feature_hash[feature_type] << context[ix].map { |e| e.to_s() }.join("#").gsub(/\s/, "_")
531:         end
532:       }
533:     }
534:   end

compute syntax-dependent features: children (that is, dependents) of the target word, parent, and siblings.

format: 

 feature type is CH for children, PA for parent, SI for siblings

 individual features are: 
   <dependency>#<word>#<lemma>#<pos>#<ne>

548:                            sent,       # SalsaTigerRegXML object
549:                            feature_hash) # hash: feature_type -> array:feature, enter features here
550:     
551:     target = sent.terminals().detect { |t| t.id() == main_target_id }
552:     unless target
553:       $stderr.puts "Featurization error: cannot find target with ID #{main_target_id}, skipping."
554:       return
555:     end
556:

558:     # compute dependencies using delegate object for grammatical functions.

561:       @current_sent = sent

570:     
571:     # parent
572:     feature_hash["PA"] = get_parents(target).map { |rel, node|
573:       #print "\t", rel, " ", node, "\n"

576:     }
577:     
578:     # siblings
579:     feature_hash["SI"] = Array.new()

584:       #print "\t", rel, " ", node, "\n"
