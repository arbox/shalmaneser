RosyEval

This class is a subclass of the general evaluation class Eval, which makes evaluation results readable via readable object variables

step: can be argrec, arglab, onestep, as usual, but also 

      - "all":
         evaluate argrec and arglab together.
         When argrec == NONE, use the argrec value, else use the arglab value
      - "prune": 
         evaluate the pruning column as if it were an argrec assignment

When step == argrec or prune, evaluate only the target class FE Otherwise, evaluate all target classes
