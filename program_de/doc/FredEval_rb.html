FredEval Katrin Erk April 05

Frame disambiguation system: evaluate classification results

While the other main classes of Fred just provide a new() method and a compute() method, the FredEval class also provides access methods to all the individual evaluation results and allows for a flag that suppresses evaluation output to a file — such that this package can also be used by external systems that wish to evaluate Fred.

Inherits from the Eval class that is not Fred-specific
