WhereWasI

WhereWasI

keep track of a lengthy computation such that, if it is interrupted, we can start again where we left off

Manages a list_of_steps, an array of strings. the calling process notifies the WhereWasI object when one of the steps is finished. Then the step is logged in a logfile.

If, on starting the WhereWasI object, the logfile is found to exist, the process is taken up after the last step recorded in the logfile.

the WhereWasI object keeps a list of steps (strings) and offers an access method next_step() that returns the next step to be taken.
