each_group

iterates through the “xwise” groups, sets internal values such that get_a_view_for_current_group() will get you the correct view

yields: for each group, a pair of

  • the hash describing the group, as returned by unique_values_of_column

  • plus an ID for the group, made up of its hash values concatenated into a string (values are connected by spaces)
